# debloat Huawei tablet
script that debloat a Huawei device. Remove bloat services as well as telemetry. Keep only the core services so it can work.

## /!\ Disclaimer
I do not hold any responsibility if your device gets bricked.
I created this script solely for Huawei MediePads.
It may be used on other Huawei devices but I can't confirm that it will work.

## how to use
The script works only on Linux.
This require adb.
adb may be installed by using the folloiwing command : ``sudo apt install android-tools-adb``
Once its installed, git clone the project or download it.
You may need to chmod the script.
connect you Huawei device. Unlock the developer settings, activate USB debugging.
Connect the tablet to your pc, launch the script, let it do its magic.
It's done.
